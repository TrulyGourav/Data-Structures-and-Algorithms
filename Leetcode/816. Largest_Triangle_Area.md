# 812. Largest Triangle Area --- Complete Documentation

## 📝 Problem Statement

Given an array of 2D points `points[i] = [xi, yi]`, return the **area of
the largest triangle** that can be formed using **any three distinct
points**.

The answer must be accurate within **1e-5**.

------------------------------------------------------------------------

## 🌟 Intuition (Most Important)

To find the area of any triangle formed by three 2D points **A(x1,
y1)**, **B(x2, y2)**, **C(x3, y3)**, we can use the **Shoelace Formula /
Determinant Formula**:

\[ `\text{Area}`{=tex} = `\frac{1}{2}`{=tex}
`\cdot `{=tex}\|(x_2-x_1)(y_3-y_1) - (x_3-x_1)(y_2-y_1)\| \]

This gives the **signed area** (parallelogram area), so taking half +
absolute value gives the triangle area.

Since constraints are small (max 50 points), we can simply check **every
triplet**.

------------------------------------------------------------------------

## 🔍 Conceptual Understanding

-   The determinant gives twice the area of the triangle.\
-   Using 3 nested loops is fine because max combinations = C(50,3) =
    19,600 → very small.\
-   Geometry problems often reduce to formula applications like this.

## 📐 Visual Recursion/Geometry Understanding

Consider 3 points:

         C(1,2)
          *
         / \
        /   \
    A(0,0)*---* B(2,0)

Area = 1/2 × base × height = 1/2 × 2 × 2 = **2**\
This matches the determinant formula.

------------------------------------------------------------------------

## ✅ Final Code (Given Solution)

``` java
class Solution {
  public double largestTriangleArea(int[][] points) {
    double ans = 0;

    for (int[] A : points)
      for (int[] B : points)
        for (int[] C : points)
          ans = Math.max(ans, 0.5 * Math.abs((B[0] - A[0]) * (C[1] - A[1]) -
                                             (C[0] - A[0]) * (B[1] - A[1])));

    return ans;
  }
}
```

## 🎯 Memory Trick (Never Forget the Logic)

Think of the formula like this:

👉 **"Base × Height but hidden inside vectors."**\
The cross-product `(x2-x1)(y3-y1) - (x3-x1)(y2-y1)` is literally the
**area of the parallelogram** formed by vectors **AB** and **AC**.

So:

-   Cross product = parallelogram\
-   Half = triangle\
-   Absolute value = area

🔑 **Mnemonic:**\
💡 *"Cross → Parallelogram → Half → Triangle"*

Whenever you see 3 points → immediately think:\
➡️ **Cross product gives triangle area in disguise.**

This will make geometry area problems effortless in interviews.
