# 2110. Number of Smooth Descent Periods of a Stock

**Difficulty:** Medium\
**Platform:** LeetCode\
**Pattern:** Contiguous Subarray + Counting DP Pattern

------------------------------------------------------------------------

## 🧠 Problem Statement

You are given an integer array `prices` representing the daily price
history of a stock, where `prices[i]` is the stock price on the ith day.

A smooth descent period consists of one or more contiguous days such
that:

-   The price on each day is lower than the previous day by **exactly
    1**
-   The first day of the period is exempted from this rule.

Return the total number of smooth descent periods.

------------------------------------------------------------------------

## 📌 Example

### Example 1

Input:

    [3,2,1,4]

Output:

    7

Explanation: Valid periods: \[3\], \[2\], \[1\], \[4\], \[3,2\],
\[2,1\], \[3,2,1\]

------------------------------------------------------------------------

## 🌟✨ INTUITION (Most Important Section)

Think of this as:

👉 "For every index, how many smooth descent subarrays END at that
index?"

If current element continues the smooth descent
(i.e. `prices[i] == prices[i-1] - 1`),\
then we can extend all previous descent sequences.

Otherwise, we start fresh from that index.

So instead of generating all subarrays (which is O(n²)),\
we simply count how many valid subarrays end at each position.

🔥 Key Realization: If a descent streak length is `k`,\
number of smooth subarrays inside it = 1 + 2 + 3 + ... + k

But instead of formula, we accumulate dynamically.

------------------------------------------------------------------------

## 🔍 Conceptual Understanding

Let: - `dp` = length of current smooth descent streak ending at index
i - `ans` = total smooth descent periods

At every index: - If descent continues → `dp++` - Else → `dp = 1` - Add
`dp` to answer

Why? Because dp tells how many subarrays end at current index.

## 🌳 Visual Understanding (Small Test Case)

Example:

    prices = [3, 2, 1, 4]

Step by Step:

Index 0: 3 → dp = 1 → ans = 1

Index 1: 2 = 3-1 → dp = 2 → ans = 3

Index 2: 1 = 2-1 → dp = 3 → ans = 6

Index 3: 4 ≠ 1-1 → dp = 1 → ans = 7

Final Answer = 7

Interpretation: At index 2 (value 1), dp=3 means: Subarrays ending at 1:
\[1\] \[2,1\] \[3,2,1\]

------------------------------------------------------------------------

## 💻 Code Solution (Optimal)

``` java
class Solution {
  public long getDescentPeriods(int[] prices) {
    long ans = 1; // prices[0]
    int dp = 1;

    for (int i = 1; i < prices.length; ++i) {
      if (prices[i] == prices[i - 1] - 1)
        ++dp;
      else
        dp = 1;
      ans += dp;
    }

    return ans;
  }
}
```

👉 Think: "How many valid subarrays END at each index?"

This converts O(n²) → O(n).
